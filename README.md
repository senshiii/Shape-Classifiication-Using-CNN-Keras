# Shape-Classifiication-Using-CNN-Keras
A Convolutional Neural Network trained to classify filled shapes of four types [ 'Square', 'Triangle', 'Circle', 'Star' ]

## Layers used in CNN:
- 2d Convolution Layer
- Max Pooling Layer
- 2d Convolution Layer
- Max Pooling Layer
- Flatten Layer
- Output Layer
